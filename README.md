
## Laravel Encryption&Decryption app;

> **_NOTE:_**  to encrypt and decrypt files


## Project Setup


- 1-  create vendor folder.
> composer update;
 
- 2-  Copy .env.example to .env:
> copy .env.example .env

- 3-  Generate a key:
> php artisan key:generate


- 4- link storage 
>  php artisan storage:link


- 5- run the project
> php artisan serve
